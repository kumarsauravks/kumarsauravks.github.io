C# QnA

Q1. Explain difference between .NET and C# ?.NET Framework and .NET core ?

.Net vs C#

  • .Net is a framework that contains all the libraries required to execute a program.
  • C# is a programming language which is used to write code.

.NET Framework vs .NET core

  • .NET Framework runs only on Windows. .NET Core is cross platform.
Q2. What is IL? What is the use of JIT? Is it possible to view IL code? Benefit of compiling into IL code?

IL (Intermediate Language) Code

  • IL Code is a partially compiled code.
  • JIT compiles IL code to Machine language.
  • We can see IL code with the help of assemblers.
  • The runtime environment and development environment can be very different. So depending on the runtime environment JIT picks the best optimized code as per that environment.
Q3. What is CLR ( Common Language Runtime) ?

CLR

  • CLR invokes JIT to compile to IL code.
  • Cleans any unused objects by using GC.
Q4. What is managed and unmanaged code ?

Managed and Unmanaged code ?

  • Code that executes under CLR execution environment is called as managed code.
  • Unmanaged code executes outside CLR boundary.
  • Unmanaged code is nothing but code written in C++ , VB6 , VC++ etc.
  • Unmanaged codes have their own environment in which the code runs and it's completely outside the control of CLR.
Q5. Explain the importance of Garbage collector ?

Garbage collector

  • Garbage collector automatically free up memory space that has been allocated to objects no longer needed by the program.

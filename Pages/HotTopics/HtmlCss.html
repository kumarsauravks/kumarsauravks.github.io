CSS Flexbox

Flexbox
  • Used to design flexible responsive layout structure
Flexbox Container
  • To use flexbox we need a parent div to have the property of display:flex;
  • Example : Below, is a flex container with three flex items.
  • <div class="flex-container">
      <div>1</div>
      <div>2</div>
      <div>3</div>
    </div>

    <style>
    .flex-container {
      display: flex;
      background-color: tomato;
    }
    .flex-container > div {
      background-color: #f1f1f1;
      margin: 10px;
      padding: 20px;
      font-size: 30px;
      width: 100px;
    }
    </style>
  • Output
    1
    2
    3
Flex Direction
  • flex-direction : column, column-reverse, row, row-reverse
  • Example : flex-direction: column;
  • <div class="flex-container">
      <div>1</div>
      <div>2</div>
      <div>3</div>
    </div>

    <style>
    .flex-container {
      display: flex;
      background-color: tomato;
      flex-direction: column;
    }
    .flex-container > div {
      background-color: #f1f1f1;
      margin: 10px;
      padding: 20px;
      font-size: 30px;
      width: 100px;
    }
    </style>
  • Output : flex-direction: column;
    1
    2
    3
  • Output : flex-direction: column-reverse;
    1
    2
    3
  • Output : flex-direction: row;
    1
    2
    3
  • Output : flex-direction: row-reverse;
    1
    2
    3

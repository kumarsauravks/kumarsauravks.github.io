Angular 18

Setup + Project Creation
  • NodeJS Install https://nodejs.org/en
  • Check If Node Installed npm --version
  • Install Angular Cli npm install -g @angular/cli
  • Check If Installed ng v
  • Angular Home https://angular.dev
  • Create Project ng new angular_18_tutorial
  • run project ng serve ng serve --o ng serve --port 4204
File Structure
  • package.json :-
    dependencies jo project me use krenge wo yha define krte hain.
  • angular.json :-
    angular k architecture/configuration ka file
  • style.css :-
    global style k liye
  • gitignore :-
    system related files jo mujhe push nhi krna hai. Folders / k sath aayenge eg: /node_modules and files direct naam se eg: npm-debug.log
  • tsconfig.json :-
    typescript k compilation k settings yha defined hoti hain.
  • index.html :-
    jo pura application contain krta hai hmesha run hota rahega.
  • main.ts :-
    application jab start hota hai to sabse Pehle main.ts
  • app.config.ts :-
    app.module.ts Pehle thi jo ab nhi hai because sare component ab hai standalone, use module ki jarurat ni hai.
    yha kuch cheeje jaise services routes define kr skte hain.
